# 🛒 RetailMart ETL Pipeline

An end-to-end **ETL (Extract, Transform, Load)** project built using **Python, Pandas, SQLAlchemy, SQLite, and SQL**. The pipeline extracts retail sales data from multiple CSV files, transforms it into a clean dataset, stores it in a SQLite database, and performs SQL-based analysis.

## 🚀 Features

- Extract data from multiple CSV files
- Clean and transform data using Pandas
- Merge sales, product, and store datasets
- Calculate total revenue
- Load processed data into SQLite
- Perform SQL analysis
- Generate summary reports

## 🛠️ Tech Stack
